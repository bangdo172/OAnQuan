
FILE TRONG GAME
1. File box.cpp; box.hpp: xử lí struct box: Trước em định làm class nhưng thấy khó quá giờ chuyển sang struct thôi cho dễ dùng
2. File Game.cpp; game.hpp: chạy vòng gameLoop, logic game các thứ (ở đây có 1 số hàm em gọi tên ra nhưng chưa viết được)
3. File showGraphic.cpp; showGraphic.hpp: các khởi tạo đồ hoạ SDL; các hàm showHand, showStone, showArrow và hàm showGraphic là hàm chứa cả 3 hàm (showHand, showStone, showArrow)
